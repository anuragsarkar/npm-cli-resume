# jscodelover

This is my resume as a cli application.

## Installation

- Run `npm i --global jscodelover-resume`.
- Then in terminal run `jscodelover-resume`.

### :white_check_mark: Options

As of now there are below options:

- About me
- Professional Experience
- Tech Skills
- Projects
- Personal Interest
- Contact me
